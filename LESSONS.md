# Lessons

## 2026-04-07
- A general-audience one-pager loses trust when investor material, election coverage, and civic action all compete in the same top-level scroll.
- "Commonfolk" pages need a single promised outcome on the first screen. In this case: what standards should communities demand before approving AI infrastructure.
- Progressive disclosure works better than long uninterrupted density. Specialist content is safer in appendix panels than in the primary narrative.
- On dark editorial pages, small light-weight body text feels refined in code but tiring in practice. Use stronger contrast and a real reading size.
- Charts and educational tools should not depend on hover or custom non-native controls for core understanding.
